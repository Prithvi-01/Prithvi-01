## 🏆 Featured Open-Source Projects

### [1. NexusAI: Enterprise Multi-LLM Orchestration & RAG Platform](https://github.com/Prithvi-01/nexusai)
A cost-free, entirely local, production-grade AI platform. Designed for high-throughput prompt routing using a custom offline Semantic Cache (sub-5ms responses). It deploys onto AWS Free-Tier using fully automated Terraform Infrastructure-as-Code.
- **Stack:** Python, FastAPI, Next.js 14, SQLite, ChromaDB, SentenceTransformers, Ollama, Docker, AWS
- **Highlights:** Dynamic LLM Router (Reasoning, Coding, Extraction), Sub-Millisecond Semantic SQL Cache, Asynchronous RAG Chunking Pipeline.

### [2. ME-UP: AI-Powered Hyperlocal Creator Marketplace](https://github.com/Prithvi-01/ME-UP)
An advanced intelligent commerce ecosystem built for independent creators and small businesses. ME-UP eliminates keyword search by implementing deep semantic discovery ("Find homemade spicy biryani near me").
- **Stack:** Flutter, FastAPI, Python, PostgreSQL, Supabase, Meilisearch, OpenAI APIs.
- **Highlights:** AI Product Discovery, Dynamic Recommendation Engines, Generative Product Pages, AI Chat Assistants.
