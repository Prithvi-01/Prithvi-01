## ✦ Core Projects

### 🚀 [NexusAI](https://github.com/Prithvi-01/nexusai) - Enterprise Multi-LLM Orchestration
NexusAI is a cost-free, production-grade AI platform designed from the ground up for advanced prompt routing, sub-millisecond semantic caching, and local Retrieval-Augmented Generation (RAG). 
- **The Intelligence Engine:** Implements a hybrid intent classification engine that dynamically routes queries to the most optimal local model (e.g., reasoning to Llama3, coding to Mistral).
- **Sub-Millisecond Cache:** Utilizes normalized SQLite vector representations to bypass LLM inference entirely for highly similar prompts, delivering responses in under 5 milliseconds.
- **Tech Stack:** Python, FastAPI, Next.js, ChromaDB, SentenceTransformers, Ollama, Docker, AWS EC2.

### 🌊 [FlowSpace AI](https://github.com/Prithvi-01/flowspace-ai) - Intelligent Google Workspace Command Center
FlowSpace AI transforms the standard browser experience into a highly productive, AI-driven liquid-glass dashboard. 
- **The Core Architecture:** Seamlessly overrides the default Chrome "New Tab" page to act as a centralized, AI-enhanced command center.
- **Smart Integrations:** Provides deep, read-only AI integrations with the Google Workspace ecosystem (Gmail, Calendar, Drive metadata, and Tasks) to offer immediate workflow insights.
- **Tech Stack:** React 18, TypeScript, Tailwind CSS, Framer Motion, Vite, Manifest V3.

### 🛍️ [ME-UP](https://github.com/Prithvi-01/ME-UP) - Hyperlocal AI Creator Marketplace
ME-UP is an advanced AI-powered ecosystem designed to empower home-based creators, independent artists, and cooks by intelligently matching their unique products to local consumers.
- **Intelligent Discovery:** Employs embedding-based semantic search and Retrieval-Augmented Generation (RAG) to process complex consumer queries and match them with highly tailored, hyperlocal products.
- **Creator Empowerment:** Features sophisticated AI mechanics to automate store metadata, descriptions, and dynamic product recommendations without requiring a physical storefront.
- **Tech Stack:** Flutter, Python, FastAPI, Supabase, PostgreSQL, OpenAI.
